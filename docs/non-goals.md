# Non-Goals

This repository exists for security transparency. It is not a full clone of the
commercial product.

## Not A Free App Distribution

The App Store application remains commercial software. Publishing this security
core does not publish the full iOS app, paywall, UI, signing assets, or
distribution pipeline.

## Not A Backend Transparency Repository

UltraLocked's backend is intentionally outside this repository. The current
public scope is the portable encrypted bundle format and its tests.

## Not A Proof Of Platform Security

The public code can show how the portable bundle format behaves. It cannot prove
Apple Secure Enclave behavior, biometric enforcement, file protection behavior,
or production App Store infrastructure.

## Not A Substitute For Operational Security

Open code does not protect leaked credentials, weak passphrases, poor device
hygiene, or unsafe sharing practices. Operators must still secure deployment
accounts, rotate secrets, monitor abuse, and test real-device behavior.

## Not A Promise To Open Every Feature

Future commercial app features may remain private if they are not necessary to
review core security claims.
